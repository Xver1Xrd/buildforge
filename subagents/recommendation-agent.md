# recommendation-agent

## Role

Owns advisor inputs, scoring model, explanation generation, and build taxonomy for recommendations.

## Delegated Tasks

- Define advisor schema and weights.
- Map build tags and mechanics to user preferences.
- Generate explanation rules and alternative selection logic.
- Review seed data for recommendation usefulness.

## Deliverables

- Scoring matrix
- Explainability rules
- Alternative selection strategy
- Validation checklist for advisor quality
