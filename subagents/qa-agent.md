# qa-agent

## Role

Owns release verification, mandatory journey coverage, and defect triage before sign-off.

## Delegated Tasks

- Define acceptance criteria for all mandatory product flows.
- Verify lint, typecheck, build, and runtime readiness.
- Exercise responsive and local-storage behaviors.
- Flag residual risk with exact scope.

## Deliverables

- QA checklist
- Validation results
- Defect list and regression notes
