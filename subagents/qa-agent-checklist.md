# qa-agent Checklist

- Home page loads with both supported games and featured builds.
- Game landing and catalog pages browse seeded content without empty states.
- Build detail pages render all required sections for each game.
- Compare page highlights meaningful differences for selected builds.
- Advisor returns a best match and two alternatives with explanations.
- Favorites persist locally across refresh.
- Dashboard renders chart and summary data.
- `lint`, `typecheck`, and `build` must pass before sign-off.
