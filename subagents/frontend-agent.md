# frontend-agent

## Role

Owns visual system, responsive composition, theme variance by game, and interaction polish.

## Delegated Tasks

- Design shared UI primitives and game-specific variants.
- Build premium page compositions for home, catalog, detail, compare, advisor, and dashboard.
- Keep motion restrained and purposeful.
- Protect accessibility and responsive behavior.

## Deliverables

- Theme token plan
- Component variant plan
- Page composition decisions
- Final UI polish pass
